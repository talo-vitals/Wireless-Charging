## How to Run Tests

In PlatformIO:
- Click **"Test"** button in the bottom bar
- Or run: `platformio test`

For more information, see the [PlatformIO Testing Documentation](https://docs.platformio.org/en/latest/core/userguide/cmd_test.html).
