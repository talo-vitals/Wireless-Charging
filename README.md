# Wireless-Charging
IoT wireless charging system with automatic power control using current sensing.

## Features
- Automatic device detection usung ACS712 current sensor
- Reley control to save power when not charging
- Live current monitoring
- Web interface for remote control
- Real-time status display on TFT screen

  ## Hardware Used
  - ESP32 Devkit with 1.8" Touch screen
  - 5v 2a Wireless Charging Modules
  - ACS712 Current Sensor
  - 5v Relay Module
  - 5v 5A Power Supply
 
  ## Future Plans
  Scalinguo for wireless charging for robots, drones, vehicles, office desk, homes.
